## Installation

1. **Download APK file:** Find the latest version in the `final_apk` folder.
2. **Enable Installation from Unknown Sources:** Go to your device's settings, navigate to "Security" or "Biometrics and Security," and enable "Install Unknown Apps" or "Install from Unknown Sources" for your file browser.
3. **Install the App:** Open the downloaded APK file and follow the installation prompts.

## Usage

1. Open the GYST app on your Android device.
2. Explore the available planners and schedules.
3. Select the one that suits your needs and start organizing your tasks.
4. Experience enhanced productivity and achievement!
